# Self-Hosted Observability Stack

> Full visibility into every node and service.

## Overview

A monitoring platform for the homelab, with Prometheus scraping metrics across hosts and containers, Grafana dashboards for health and capacity, and alerting that fires on service degradation.

## Tech stack

- Prometheus
- Grafana
- Alertmanager
- Exporters

## What this lab demonstrates

- Metric scraping across hosts and containers
- Grafana health and capacity dashboards
- Alerting on service degradation
- Capacity and reliability visibility
